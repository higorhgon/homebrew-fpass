# Homebrew Tap for fpass

Tap Homebrew pessoal para instalar [fpass](https://github.com/higorhgon/fpass).

## Instalação

```bash
brew tap higorhgon/fpass https://github.com/higorhgon/homebrew-fpass
brew install fpass
```

## Uso

```bash
fpass
```

## Desinstalar

```bash
brew uninstall fpass
brew untap higorhgon/fpass
```
